# 贡献者行为准则 · Code of Conduct

## 我们的承诺

为营造开放、友善的社区环境，我们作为贡献者和维护者承诺：让每个人参与本项目和社区的体验都免于骚扰，无论其年龄、体型、身体健全与否、族裔、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向。

## 我们的准则

有助于营造积极环境的行为包括：

- 使用友善、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 以社区利益为重
- 对其他社区成员保持同理心

不可接受的行为包括：

- 使用与性相关的言语或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱/贬损性评论，人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，发布他人的私人信息（如住址、邮箱等）
- 在专业场合中其他被合理认定为不当的行为

## 我们的责任

项目维护者有责任明确可接受行为的标准，并对任何不可接受的行为采取恰当、公正的纠正措施。

维护者有权删除、编辑或拒绝不符合本准则的评论、提交、代码、Issue 及其他贡献。

## 适用范围

本准则适用于所有项目空间，以及个人代表项目或社区出现在公共场合的场景。

## 执行

如遇滥用、骚扰或其他不可接受的行为，可联系项目维护者报告。所有投诉都将被审查和调查，并作出适当且必要的回应。维护者有义务为报告者保密。

## 归属

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 1.4 版。
