## 一、场景-功能-价值映射总览

### 1.1 供应链管理全场景覆盖

| 编号 | SMIC业务场景 | 涉及物料 | 当前痛点 | 智衍功能模块 | 对齐度 | 价值量化 |
|:----:|:------------|:---------|:---------|:------------|:-----:|:--------|
| S1 | **28nm产线物料齐套管理** | 硅片/光刻胶/特气/靶材 | 人工检查耗时4h/次，发现缺料时已影响排产 | Agent齐套检查 | ⭐⭐⭐⭐⭐ | 效率提升360倍，缺料发现提前12h |
| S2 | **光刻胶库存预警** | ArF/KrF/i-line光刻胶 | 进口依赖高，交期45天+，过期报废损失大 | 库存监控+预警 | ⭐⭐⭐⭐⭐ | 降低过期报废80%+ |
| S3 | **特种气体供应保障** | NF3/WF6/高纯N2/Ar | 危化品管理严格，断供风险高 | 替代推荐+PO跟踪 | ⭐⭐⭐⭐ | 替代方案响应从天级降到分钟级 |
| S4 | **靶材采购优化** | Ti/Cu/Al靶材 | 高单价(3000-4500元)，供应商集中 | 替代推荐+价格比较 | ⭐⭐⭐⭐ | 采购成本优化5-15% |
| S5 | **国产替代导入评估** | 光刻胶/气体/靶材 | 国产供应商验证周期长，缺乏量化评估 | 替代引擎+国产标签 | ⭐⭐⭐⭐⭐ | 国产替代导入效率提升 |
| S6 | **设备备件补货** | 石英窗片/O型圈/聚焦环 | 独家供应，交期60-90天 | 自动补货建议 | ⭐⭐⭐ | 减少设备待料停机 |
| S7 | **多品类供应风险总览** | 全部品类 | 缺乏统一风险视图 | 事件通知+预警面板 | ⭐⭐⭐ | 全局风险一目了然 |

## 四、对接SMIC的准备工作

### 4.1 数据接口准备

| SMIC系统 | 需要对接的数据 | 智衍对接方式 | 预估工作量 |
|:---------|:-------------|:-----------|:---------:|
| SAP/ERP | BOM、PO、物料主数据 | MCP协议适配器 | 3-5天 |
| WMS | 库存余额、库位 | MCP协议适配器 | 2-3天 |
| SRM | 供应商数据、评级 | MCP协议适配器 | 2-3天 |
| MES | 在制品(WIP)数据 | 暂不接入(MVP) | — |

### 4.2 边缘部署方案

```
SMIC内网                   智衍容器
┌────────────┐          ┌──────────────────┐
│ SAP/ERP    │─→API──→  │  Runtime (FastAPI)│
│ WMS        │─→API──→  │  + PostgreSQL    │
│ SRM        │─→API──→  │  + 元Agent        │
│ 数据源     │          │  + Agent Engine   │
└────────────┘          └──────────────────┘
                               │
                          ┌────┴────┐
                          │ Studio  │ ← 供应链团队浏览器访问
                          │ (内网)   │
                          └─────────┘
```

**关键点**：所有组件部署在SMIC内网，数据不出厂

### 4.3 分阶段上线路径
