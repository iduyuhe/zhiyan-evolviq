## 四、演示脚本（20分钟）

### 开场（2分钟）— 场景共鸣

> "谢总，今天不是来卖软件的，是来给SMIC的供应链装一个'AI员工'。"

**话术要点**：
1. 点出晶圆厂供应链的独特挑战（30天+交期、进口依赖、高价值）
2. 理解SMIC作为中国最大晶圆厂的供应链复杂度
3. 今天要展示的：一个Agent能在40秒内完成人工4小时的工作

### 场景演示（8分钟）— 现场跑

**Step 1：设定目标**
```
操作：在Agent Studio输入框内输入
"每2小时检查28nm产线物料齐套，
 重点关注硅片和光刻胶，
 缺料风险>30%时自动检索替代方案，
 在5%价格波动内可直接锁定库存"
```
> "看，用一句话告诉Agent你要做什么。不需要培训，不需要写代码。"

**Step 2：Agent生成规划**
```
→ Agent自动展示执行规划
→ 人可以看到：Agent打算查什么数据、怎么做分析
```
> "这是AI的'工作计划'，人看一眼，觉得合理就批准，不合理就驳回。"

**Step 3：批准执行**
```
→ 点击"确认执行"
→ Agent开始：查BOM → 查库存 → 查PO → 算风险 → 找替代
→ 大约40秒后出结果
```
> "谢总，这个过程中Agent干了三件事：查了12项物料库存、
>  对比了在途PO交期、检索了替代供应商。人工做这些至少要4小时。"

**Step 4：看结果**
```
→ 展示齐套明细表
→ 展示预警（红色高亮）
→ 展示已执行的操作
```
> "这是刚才跑出来的结果——12项物料齐套率41.7%。
>  硅片和光刻胶严重缺料，Agent已经在授权范围内锁定了替代库存。
>  如果SMIC的数据接入，这个分析会更加精准。"

### Q&A准备（5分钟）

**Q：你们的Agent和传统的ERP预警有什么区别？**
A：ERP预警是静态阈值，到了才报警。我们的Agent是主动的——它会提前预判风险、检索替代方案、在授权范围内直接执行。不是"告诉你下雨了"，而是"帮你带伞"。

**Q：数据安全性怎么保障？**
A：有两种部署方式。正式使用时采用边缘部署——Agent运行在SMIC本地的服务器上，所有数据不出厂。今天的演示用的是模拟数据。

**Q：和现有的SAP/SRM怎么集成？**
A：我们不替换任何现有系统。Agent通过MCP协议对接现有系统的API接口，相当于在现有系统上叠加了一层AI决策层。

**Q：需要多长时间才能上线？**
A：如果SMIC配合提供数据接口，第一品类可以在2-4周内上线试运行。我们采用"先跑通一个品类再扩展"的策略，最小化风险。

## 六、数据安全承诺

> **智衍关于数据安全的承诺**：
> 1. ⚡ MVP阶段：使用SMIC提供的脱敏数据跑离线验证，不接入生产系统
> 2. 🔒 正式阶段：全栈边缘部署，所有数据存储和处理在SMIC防火墙内完成

> 5. 🔐 数据主权：客户数据100%归客户所有，任何时候可删除
