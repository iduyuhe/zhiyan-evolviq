### 1.3 两大趋势判断（SMIC管理层）

```
趋势一：产业链海外回流
→ 海外客户向中国本土供应链转移
→ 供应链复杂度增加，管理难度加大

趋势二：国产替代加速
→ 国内客户新产品替代海外老产品
→ 国产材料/设备导入需求旺盛
→ 供应商管理从"选型"到"培育"的转变
```

### 3.2 首品类建议：光刻胶

**选择理由**：

| 维度 | 评估 |
|:----|:------|
| **战略价值** | 光刻胶是国产替代重点品类，战略敏感度高 |
| **管理复杂度** | 3-5种、多供应商、存储有寿命限制，管理难度大 |
| **数据可得性** | 品类相对标准化，数据接口容易对接 |
| **失败成本低** | 即使出问题，不影响产线正常运行 |
| **价值可量化** | 过期报废损失可直接统计 |

## 四、差异化竞争定位

### 4.1 与现有方案的核心区别

```
传统SCM系统                   智衍 Agent
─────────                    ──────────
❌ 被动告警（到了阈值才报）     ✅ 主动预测（提前预判风险）
❌ 信息到人（发邮件通知）       ✅ 决策到行动（授权内直接执行）
❌ 单一数据源                 ✅ 多维度融合（BOM+库存+PO+供应商）
❌ 固定规则                   ✅ 动态决策（每次最优而非固定阈值）
❌ 升级=项目制（6-12月）       ✅ 新Agent=3天（可扩展）
```

### 4.2 对SMIC的独特价值

```
1. 🏭 晶圆厂行业深度适配
   └── 理解半导体物料特点（硅片/光刻胶/特气/靶材）
   └── 理解晶圆厂管理节奏（批次管理、有效期管理）

2. 🔒 数据安全天然满足
   └── 边缘部署（数据不出晶圆厂）
   └── 授权边界（Agent不能越界操作）
   └── 审计追溯（每步操作都有记录）

3. 🇨🇳 国产替代战略支撑
   └── Agent自动评估国产供应商替代可行性
   └── 支持"二供导入"决策
```

## 五、风险与对策

### 5.1 对齐过程中的风险

| 风险 | 等级 | 影响 | 对策 |
|:----|:----:|:----|:-----|
| SMIC内部对AI Agent接受度 | 🟡 中 | 可能被视为"玩具"而非生产力 | 先用真实数据跑一次，用数字说服 |
| 现有系统接口不开放 | 🟠 高 | 数据无法接入，Agent变花瓶 | 准备两种方案：API对接 / CSV导入 |
| 审批流程长 | 🟡 中 | 拖慢试用周期 | 建议"先离线验证，再在线试跑" |
| 数据安全法规限制 | 🟡 中 | 数据使用范围受限 | 边缘部署方案，所有计算在本地 |

### 5.2 关键成功因素

```
✅ 让谢总看到一个"可执行的AI"而不是"另一个报表系统"
✅ 第一次演示就要看到真实数据的价值（不是Demo数据）
✅ 找到SMIC内部的"推动者"——IT部门或供应链部门的合作方
✅ 先小范围验证（1个品类），再用结果说话扩展
```
